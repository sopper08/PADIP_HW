# README  
*PADIP means Principles and Applications of Digital Image Processing*

## My Tool  
### Concatenate images  
```sh
$ python3 long_long_image.py <image-folder>
# [ ], [ ], [ ] -> [ | | ]
```
### Convert to .jpg file  
```sh
$ python3 to_jpg.py <filename>
```
